# 局域网共享软件设计文档

## 项目概述

一款高性能的局域网文件共享工具，支持 Windows 电脑与 Android 手机之间的快速文件传输、剪贴板共享、屏幕共享和即时聊天。

---

## 技术选型

| 组件 | 技术方案 | 理由 |
|------|---------|------|
| **Windows 端** | Go + Fyne UI | 原生性能、跨平台、单二进制文件 |
| **Android 端** | Web UI (响应式) | 无需安装 APP，浏览器直接访问 |
| **设备发现** | UDP 广播 + mDNS | 自动发现局域网设备 |
| **文件传输** | TCP + 自定义协议 | 支持断点续传、加密传输 |
| **实时通信** | WebSocket | 剪贴板同步、文字聊天 |
| **安全加密** | AES-256-GCM + TLS | 传输数据全程加密 |

## 核心模块设计

### 1. 设备发现模块

**协议**: UDP 广播 + mDNS

**工作流程**:
```
1. 设备启动 → 监听 UDP 端口 53317
2. 广播心跳包 (每 3 秒)
3. 收到心跳 → 注册设备到本地列表
4. 设备离线 → 30 秒无心跳自动移除
```

**心跳包格式** (JSON):
```json
{
  "id": "设备唯一ID",
  "name": "设备名称",
  "type": "windows|android",
  "ip": "192.168.1.100",
  "port": 53317,
  "capabilities": ["file", "clipboard", "chat", "screen"],
  "timestamp": 1234567890
}
```

### 2. 文件传输模块

**特性**:
- 断点续传
- AES-256-GCM 加密
- 并行传输
- 进度回调

**传输协议**:
```
┌──────────────────────────────────────────────────────────┐
│                    文件传输流程                           │
├──────────────────────────────────────────────────────────┤
│                                                          │
│  发送方                          接收方                   │
│    │                               │                     │
│    │───── 1. 发送文件元数据 ──────►│                     │
│    │       (文件名/大小/MD5)        │                     │
│    │                               │                     │
│    │◄──── 2. 确认/请求续传偏移 ────│                     │
│    │                               │                     │
│    │───── 3. 发送加密文件块 ──────►│                     │
│    │       (每块 64KB)             │                     │
│    │                               │                     │
│    │◄──── 4. ACK 确认 ────────────│                     │
│    │                               │                     │
│    │───── 5. 传输完成 ───────────►│                     │
│    │                               │                     │
└──────────────────────────────────────────────────────────┘
```

**数据包格式**:
```
┌─────────────────────────────────────────┐
│ Header (32 bytes)                       │
├─────────────────────────────────────────┤
│ Magic: 4 bytes  (0x4C414E54 = "LANT")   │
│ Version: 1 byte                         │
│ Type: 1 byte (META/DATA/ACK/DONE/ERROR) │
│ SessionID: 8 bytes                      │
│ Sequence: 4 bytes                       │
│ PayloadLen: 4 bytes                     │
│ Checksum: 4 bytes (CRC32)               │
│ Reserved: 6 bytes                       │
├─────────────────────────────────────────┤
│ Payload: variable                       │
│ (加密后的文件数据或元数据)                │
└─────────────────────────────────────────┘
```

### 3. 共享文件夹模块

**功能**:
- 添加/移除共享目录
- 浏览远程共享文件
- 权限控制 (只读/读写)

**API 设计** (REST over HTTPS):
```
GET  /api/shares              # 获取共享列表
POST /api/shares              # 添加共享目录
GET  /api/files/{path}        # 浏览文件
GET  /api/download/{path}     # 下载文件
POST /api/upload/{path}       # 上传文件
DELETE /api/files/{path}      # 删除文件
```

### 4. 剪贴板同步模块

**同步内容**:
- 纯文本
- 图片 (Base64 编码)
- 文件路径 (触发快速传输)

**WebSocket 消息格式**:
```json
{
  "type": "clipboard",
  "data": {
    "content_type": "text|image|file",
    "content": "...",
    "timestamp": 1234567890,
    "source_device": "device-id"
  }
}
```

### 5. 文字聊天模块

**功能**:
- 设备间即时消息
- 消息历史记录
- 在线状态显示

**WebSocket 消息格式**:
```json
{
  "type": "chat",
  "data": {
    "message_id": "uuid",
    "from": "device-id",
    "to": "device-id",
    "content": "消息内容",
    "timestamp": 1234567890
  }
}
```

## 安全设计

### 加密方案

1. **传输加密**: TLS 1.3
2. **文件加密**: AES-256-GCM
3. **密钥交换**: ECDH (Curve25519)

### 设备配对流程

```
1. 设备 A 发起配对请求
2. 设备 B 显示配对码 (6位数字)
3. 设备 A 输入配对码确认
4. 双方保存对端公钥
5. 后续通信使用公钥加密
```
